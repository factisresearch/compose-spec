Docker Compose Spec TypeScript
===============================================

[![npm](https://img.shields.io/npm/v/@gfi-centre-ouest/docker-compose-spec-typescript.svg)](https://www.npmjs.com/package/@gfi-centre-ouest/docker-compose-spec-typescript)
